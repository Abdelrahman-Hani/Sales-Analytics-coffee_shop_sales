# Sales Analytics Project – Python & Power BI

## Overview
This project demonstrates an **end-to-end sales analytics workflow**, from raw data cleaning in Python to interactive dashboard creation in Power BI. The dataset includes retail transactions with details on products, customers, stores, weather, and holidays.

---

## Project Steps

### 1️⃣ Data Cleaning and Preparation (Python)
- Handled missing values and ensured data consistency.
- Processed timestamps and extracted features: hour, day, month.
- Prepared clean and structured CSV ready for Power BI analysis.

### 2️⃣ Power BI Integration
- Imported cleaned dataset into Power BI.
- Built a structured data model linking transactions, products, stores, and customer information.

### 3️⃣ Sales Analysis
- Sales by **city, country, store type, and payment method**.
- Temporal trends: **monthly, daily, and hourly sales patterns**.
- Identified peak hours and high-performing regions.

### 4️⃣ Product & Customer Insights
- Sales by **product name** and **product category**.
- Top **5 best-selling products**.
- Customer analysis: **age group, gender, loyalty membership**.

### 5️⃣ Advanced Analysis
- **Impact of holidays** on sales trends.
- **Weather conditions** and temperature effects on sales and product performance.

---

## Key Features
- Fully interactive **Power BI dashboard** with slicers for city, product, time, and customer segments.
- Dynamic **KPIs**: total sales, average sales per transaction, top products, and high-performing regions.
- Real-time filtering: all visuals update instantly based on selections.
